# React Ant Design Website

智能工单管理系统是一个基于React + Ant Design的现代化Web应用，提供工单创建、管理、追踪等功能。系统采用组件化开发，使用Vite作为构建工具，集成了React Router进行路由管理。

## Setup Instructions

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

4. 预览生产构建：
```bash
npm run preview
```
